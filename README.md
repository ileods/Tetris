# Tetris
Тетрис для двух игроков, написан на _JavaScript_ с использованием элементов _canvas_.

# GitHub Pages
<a href="https://ileods.github.io/Tetris/">Game</a>

## Управление игрой

 * D и стрелка вправо двигают элемент вправо
 * A и стрелка влево двигают элемент влево
 * W и стрелка вверх поворачивают элемент
 * S и стрелка вниз опускают элемент
 
 Первый элемент двигает элементы на первом поле, второй соответственно на втором.
 
 ## Правила
 
  * За каждую убранную линию +100 очков
  * За каждые подряд 4 убранные линии +1 тетрис
  * Уровень зависит от количества очков
  * Чем выше уровень, тем быстрее падают элементы
